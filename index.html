Ping Pong

Let's play! Enter a number and watch what happens. Every number leading up to it will be displayed! But not so fast... If any number smaller than that is divisible by 3, PING! And also if any number smaller than that is divisible by 5, PONG! And furthermore, if any number smaller than that is divisible by 3 AND 5, PING PONG!!!
